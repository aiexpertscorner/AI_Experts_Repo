# Build and Pipeline Overview

## Pipeline 1 — Taxonomy
Purpose:
Build and maintain master taxonomy maps.

## Pipeline 2 — Enrichment
Purpose:
Transform raw tool and content data into enriched master datasets.

## Pipeline 3 — Build datasets
Purpose:
Build page-specific datasets from enriched master data.

## Pipeline 4 — Audit
Purpose:
Validate data health, internal linking, slugs, coverage, output integrity.

## Pipeline 5 — Astro build
Purpose:
Render the site from the build datasets.

## Insights extension
Purpose:
Run insights/news/signals pipelines separately from the core site pipelines.
